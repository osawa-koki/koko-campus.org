スプレット構文

「...hoge」の「...」をスプレット構文と言います。
別に言葉に詰まったときに使用するのではありません。

使用目的は、配列の要素を展開します。 const fruits = ["りんご", "ばなな", "さくらんぼ"]; const foods = [fruits, "ごはん", "いも", "ピザ"]; console.log(foods); /* ↓ コンソール ↓ 0: Array(3) [ 'りんご', 'ばなな', 'さくらんぼ' ] 1: 'ごはん' 2: 'いも' 3: 'ピザ' */ これだと、foods配列の第一要素(インデックス番号は「0」)に配列が含まれてしまします。

ということで、スプレット構文を用いてみましょう♪ const fruits = ["りんご", "ばなな", "さくらんぼ"]; const foods = [...fruits, "ごはん", "いも", "ピザ"]; console.log(foods); /* ↓ コンソール ↓ Array(6) [ 'りんご', 'ばなな', 'さくらんぼ', 'ごはん', 'いも', 'ピザ' ] */ しっかりと、要素として展開されました。

因みに、スプレットは「展開する」という意味の英語です。
