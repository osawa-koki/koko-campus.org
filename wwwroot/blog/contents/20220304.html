深い配列操作(php)

深い配列(二次元配列以上)で、下位の要素を追加する際にその上位の要素が存在しない場合には自動で追加してくれます。
便利♪ $jp = array( "kanto" => array( "saitama" => [ "soka", "koshigaya", "kawaguchi" ], "tokyo" => [ "adachi", "chiyoda", "shibuya" ] ) ); $jp["kansai"]["kyoto"] = ["uji", "???"]; var_dump($jp); /* ↓ コンソール ↓ array(2) { ['kanto']=> array(2) { ['saitama']=> array(3) { [0]=> string(4) 'soka' [1]=> string(9) 'koshigaya' [2]=> string(9) 'kawaguchi' } ['tokyo']=> array(3) { [0]=> string(6) 'adachi' [1]=> string(7) 'chiyoda' [2]=> string(7) 'shibuya' } } ['kansai']=> array(1) { ['kyoto']=> array(2) { [0]=> string(3) 'uji' [1]=> string(3) '???' } } } */ これだけですが、上から順に確認しなくてok!なのは本当に助かります。
